Undergraduate Research (ME 392)

Autonomous Ackermann Car with Nav2 Stack

A fully autonomous indoor navigation vehicle built from scratch using ROS 2, NVIDIA Jetson, and the Nav2 framework.

DateDecember 2025
CourseME 392 – Undergraduate Research
AdvisorProf. Pranav Bhounsule
InstitutionUniversity of Illinois at Chicago
ROS 2 Humble SolidWorks NVIDIA Jetson Orin Nano Nav2 SLAM Toolbox Python 3D Printing (PETG-CF) VESC LiDAR Intel RealSense D435i

Abstract

This project focuses on building the hardware foundation of an autonomous Ackermann steering car. The work includes designing and assembling the chassis, developing an appropriate power system, and integrating sensors and key components such as LiDAR, camera, and NVIDIA Jetson Orin Nano. The objective is to complete a fully functional autonomous system from scratch that will support autonomous tasks. This first phase lays the groundwork for developing a machine learning model for lane detection in the next semester.

Mechanical Design & Fabrication

The mechanical design was developed by studying real passenger vehicles, with attention to steering geometry, drivetrain layout, and axle configuration. An Ackermann steering architecture with front steering and rear-wheel drive was selected.

Key components designed in SolidWorks include:

  • Front steering assembly
  • Rear axle assembly
  • Battery holder
  • Jetson mounting plate
  • Servo mount

All custom parts were fabricated using a Bambu Lab A1 Mini 3D printer with PETG carbon-fiber filament for stiffness-to-weight optimization. An early 3D-printed differential was replaced with a metal differential after durability testing.

Sensor & Power Integration

The vehicle is powered by a 2200 KV sensored brushless DC motor controlled via a VESC motor controller interfaced with the Jetson over UART. The power system uses two separate 4S 4400 mAh LiPo batteries — one for computing and one for the drivetrain — with a 5A inline fuse protecting the Jetson.

Perception sensors include:

  • 2D LiDAR — 12 m max range for SLAM and navigation
  • Intel RealSense D435i — RGB-D camera for depth and color

Steering is actuated by a 25 kg-cm high-torque servo driven through a PCA9685 controller over I2C.

ROS 2 Software Architecture

A modular ROS 2 Humble architecture integrates hardware control, state estimation, and navigation:

URDF / Xacro Model

Complete kinematic and geometric description with consistent dimensions across control, odometry, and simulation. Includes Gazebo plugin support for future sim work.

Motor Control Bridge

Converts /cmd_vel to motor duty cycles and servo angles using Ackermann geometry. Implements dead zones, timeouts, and current braking for safety.

Motor-Based Odometry

Estimates position and heading from VESC encoder tachometer counts and steering feedback using Ackermann kinematics. Publishes odom → base_link transform.

Web Teleoperation

Flask-based browser control panel for manual driving, steering, emergency stop, and system monitoring. Commands routed through /cmd_vel for Nav2 compatibility.

Autonomous Navigation with Nav2

The vehicle uses the full ROS 2 Nav2 stack configured for nonholonomic Ackermann vehicles:

LocalizationAMCL (Adaptive Monte Carlo Localization)
Global PlannerSMAC Hybrid Planner
Motion ModelReeds-Shepp (forward + reverse)
Local ControllerRegulated Pure Pursuit
MappingSLAM Toolbox (synchronous mode)
RecoverySpin, Backup, Drive-on-Heading, Wait
Velocity Smoothing20 Hz, ±0.3 m/s linear, ±1.0 rad/s angular

The SMAC Hybrid Planner generates kinematically feasible paths that respect steering constraints. Reeds-Shepp motion primitives enable the vehicle to reverse out of dead ends and reposition in tight spaces. The Regulated Pure Pursuit controller automatically reduces speed in high-curvature regions.

Results

⚙ Mechanical

Ackermann steering produced smooth turning. Metal differential eliminated gear wear. Dual-battery system prevented Jetson resets during operation.

📡 Sensors

LiDAR produced clean, dense scan data. RealSense provided synchronized RGB-D streams. Sensor frames aligned correctly in RViz via URDF transforms.

🚗 Control

Servo steering center calibrated at ~125° with symmetric response. Dead zone eliminated oscillations. Velocity smoothing ensured stable motion.

🗺 Navigation

Vehicle autonomously localized within a prebuilt SLAM map and navigated to user-defined goals while avoiding obstacles and respecting steering constraints.

Conclusion & Future Work

This project successfully demonstrated the complete design, fabrication, and integration of a real-world autonomous Ackermann-steered vehicle. Starting from a blank platform, the work encompassed mechanical design, sensor integration, low-level control, state estimation, SLAM mapping, and autonomous navigation with Nav2.

Future work will build on this foundation by incorporating CNN-based lane detection using the onboard RGB-D camera, sensor fusion for improved localization, and expanded testing in more complex environments.
